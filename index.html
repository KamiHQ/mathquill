mathquill by laughinghan

Render LaTeX math as pure HTML/CSS that is WYSIWYG editable

HTML and CSS are now mature enough that generating images with server-side scripts is no longer the best way to embed LaTeX-quality mathematical expressions and equations in web pages. The LatexLive? JavaScript? library not only efficiently renders LaTeX math as HTML, but to also, showcasing the power of the HTML DOM, allows real-time WYSIWYG editing of the embedded math. We have a working demo here: http://www.ocf.berkeley.edu/~hanwei/demo.html (Please note that this is an alpha version, so bugs and unimplemented features are all over the place.)

Dependencies

Powered by jQuery, v1.4+ required. Get it from http://jquery.com.
