MathQuill by laughinghan and jayferd

Render LaTeX math as pure HTML+CSS that is WYSIWYG editable

HTML and CSS are now mature enough that generating images with server-side scripts is no longer the best way to embed LaTeX-quality mathematical expressions and equations in web pages. The MathQuill JavaScript library not only efficiently renders LaTeX math as HTML, but also, showcasing the power of the HTML DOM, allows real-time WYSIWYG editing of the embedded math.

Check out our working demo!

See also: our sister project Lolgebra, online chatrooms with MathQuill that let you “lol” in math!

Usage

(Please note that this is an alpha version, so bugs and unimplemented features are all over the place.)

All you need is a script include in your header:

<script src="http://laughinghan.github.com/mathquill/mathquill.js"></script>

and wherever you’d like to have an editable math textbox:

<span class="mathquill-editable"></span>

or to convert LaTeX math to HTML:

<span class="mathquill-embedded-latex">\frac{d}{dx}\sqrt{x}</span>

Note that for dynamically created elements, you will need to call our jQuery plugin after inserting into the visible HTML DOM:

$('<span>\sqrt{e^x}</span>').appendTo('body').mathquill() or .mathquill('editable')

Any element that has been MathQuill-ified can be reverted:

$('.mathquill-embedded-latex').mathquill('revert');
